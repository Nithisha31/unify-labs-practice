Shop Manager: Inventory Control

Learning Objectives

  • Use $inc to update stock levels for all products
  • Update category names using $set and updateMany()
  • Use $push to add 'new-arrival' tag
  • Use deleteMany() to remove zero stock items

Mass Update Operations

// Increase price of all Electronics by +10
db.products.updateMany(
  { category: "Electronics" },
  { $inc: { price: 10 } }
)

// Set featured: true for items priced > 500
db.products.updateMany(
  { price: { $gt: 500 } },
  { $set: { featured: true } }
)

// Update category name example
db.products.updateMany(
  { category: "Clothing" },
  { $set: { category: "Apparel" } }
)

// Add 'new-arrival' tag to Furniture
db.products.updateMany(
  { category: "Furniture" },
  { $push: { tags: "new-arrival" } }
)

Cleanup & Verification

// Delete products with zero stock
db.products.deleteMany({ stock: 0 })

// Verify remaining documents
db.products.countDocuments()
